With Camazee plugin you can easily add video chat into your website.

== Description ==
Add option for visitors of your website to contact you using video chat directly from your wordpress website. 
Few notes about Camazee plugin
* It is easy to setup and no need to install streaming server.
* Can be used to make conference call with multiple visitors at the same time
* 2-way video/audio call 
* Text chat option with emoticons/smileys


Can be used for following type of business:
1) Online webinar
2) Helpdesk from the website
3) Private video chat
4) Online tutoring

== Installation ==

1) Download and install plugin
2) Activate the plugin at Wordpress admin panel -> Plugins
3) Follow instruction provided on the plugin menu
